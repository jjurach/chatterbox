## Implementation Details

### Service Type and Naming

- **Service Type:** `_chatterbox._tcp.local.`
- **Service Name:** `Chatterbox.<hostname>._chatterbox._tcp.local.`
- **Properties:**
  - `version`: API version (e.g., "1.0")
  - `api_path`: `/conversation` (standard path for all requests)

### Error Handling

- **Startup errors:** Non-fatal. Logs error message but doesn't block application startup.
- **Shutdown errors:** Non-fatal. Logs error message but ensures `registered` flag is set to `False`.
- **Network errors:** `get_local_ip()` raises `RuntimeError` with descriptive message; caught by `start()` and wrapped.

### Thread Safety

Zeroconf library is thread-safe for registration/unregistration. FastAPI lifespan handler is sequential (startup before any requests, shutdown after all requests complete), so no concurrency concerns.
